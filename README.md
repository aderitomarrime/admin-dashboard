## Overview
This project is a dashboard website focused on layout structure and interface organization. The website is built using HTML and CSS, with CSS Grid as the primary layout system to define the overall structure, including sidebar, header, and main content areas.


## Live Demo

Visit the website: [Live Demo](https://aderitomarrime.github.io/admin-dashboard/)


## Features

- Minimal interface  
- Grid Layout 
- CSS Custom Properties  
- Web Font Integration  


## Technologies Used

- HTML5  
- CSS3 


## Installation

To run this project locally, follow these steps:

1. Clone the repository:

```sh
git clone https://github.com/aderitomarrime/admin-dashboard.git
```

2. Navigate to the project 
```sh
cd admin-dashboard
```
3. Open index.html in your browser


## How to use

Once the website is open in your browser, simply navigate through the dashboard and explore the different sections available. Interact with the header, sidebar and main content panels to observe the layout structure, spacing, and overall interface organization.
